# Support and Questions

If you need help contributing:

## Use These Paths

- ask on the relevant GitHub issue
- ask on the relevant PR
- open a documentation or onboarding issue if the process is unclear
- for private conduct or safety concerns, contact the repository owner privately through the GitHub profile contact path instead of a public thread

## What To Include When Asking

- what you were trying to do
- what file or lane you were working in
- what command or step failed
- what you already checked

## Best Practice

Ask early when the blocker affects:

- scope
- ownership
- auth
- infra
- contracts
- release behavior

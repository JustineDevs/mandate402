# Contribution Types

Mandate402 accepts more than code contributions.

## Contribution Areas

- code
- documentation
- design / UI/UX
- testing and QA
- issue triage
- workflow and process improvements
- release-note clarity

## Non-Code Contributions Matter

Helpful contributions include:

- improving docs
- improving onboarding clarity
- improving issue quality
- identifying workflow friction
- spotting confusing UI states

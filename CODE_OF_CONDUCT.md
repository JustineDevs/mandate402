# Code of Conduct

## Our Commitment

We want Mandate402 to be a respectful, safe, and constructive project for contributors, reviewers, and maintainers.

All participants are expected to:

- be respectful
- assume good faith
- give constructive feedback
- stay focused on the work, not personal attacks
- help make the project approachable for new contributors

## Expected Behavior

- Use welcoming and respectful language.
- Accept feedback and disagreement without hostility.
- Ask questions when context is unclear instead of escalating tension.
- Be clear and honest about blockers, uncertainty, and mistakes.
- Respect ownership boundaries while still collaborating openly.

## Unacceptable Behavior

- harassment
- personal attacks
- intimidation
- discriminatory language or behavior
- repeated hostile or dismissive conduct
- deliberately wasting reviewer or contributor time through bad-faith interaction

## Enforcement

Project maintainers may remove comments, close threads, restrict participation, or reject contributions that violate this Code of Conduct.

## Reporting

If a contributor feels unsafe or sees behavior that violates this Code of Conduct, contact the repository owner privately through the GitHub profile contact path instead of opening a public thread.
